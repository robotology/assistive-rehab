Description

This module retrieves the selected 3D skeleton from objectsPropertiesCollector and sends the upper body 2D keypoints (x,y) (ankles and knees discarded), observed in a predefined window, to a pretrained LSTM. The retrieved skeleton is normalized and keypoints are transformed and rescaled by a factor of 10 (same operations were performed during training). When the window is full, the network outputs the label of the predicted action. Predicted actions are accumulated for a (settable) number of times, and the label of the most voted action is provided as output. The adopted classes are defined in the config.ini file. It makes use of TensorFlowCC.

Parameters

  • –name: Module's name; all the open ports will be tagged with the prefix /name.
  • –predict: To enable prediction.
  • –general::num-classes: Number of classes. This has to be the same as defined during training.
  • –general::num-features: Number of features (upper body 2D keypoints' coordinates) provided to the network. This has to be the same as defined during training.
  • –general::num-steps: Number of times predicted actions are accumulated. Only the most voted action is provided as output.
  • –class-i::label: Label of the class.
  • –class-i::value: Class number as provided by the network.

Input Ports

Output Ports

  • /actionRecognizer/target:o [Bottle] [default carrier:]: Outputs a yarp bottle containing the label of the action to perform, the label of the predicted action and the confidence score.
  • /actionRecognizer/opc [rpc] [default carrier:]: Sends commands to objectsPropertiesCollector to retrieve 3D skeletons. (Note: only 2D keypoints are used).
