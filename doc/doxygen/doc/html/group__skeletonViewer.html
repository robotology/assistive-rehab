Description

This module is responsible for displaying in real-time multiple skeletons in 3D. It makes use of VTK.

Parameters

  • –x: Viewer's x position in screen coordinates pixels.
  • –y: Viewer's y position in screen coordinates pixels.
  • –w: Viewer's width in screen coordinates pixels.
  • –h: Viewer's height in screen coordinates pixels.
  • –gc-period: Periodicity of the module (s)
  • –bg-color: Viewer's background color using rgb color specification.
  • –camera-position: The camera position in world coordinates.
  • –camera-focalpoint: The camera focal point in world coordinates.
  • –camera-viewup: The camera viewup direction in world coordinates.
  • –show-floor: Switch it on to display the floor.
  • –floor-center: The floor center in world coordinates.
  • –floor-normal: The floor normal in world coordinates.
  • –grid-resolution: Grid resolution of the floor expressed in number of squares per side. Since the floor size is 10x10 square meters, a resolution of e.g. (100 100) will create a grid of squares with side length 10 cm.

Input Ports

  • /skeletonViewer:i [Bottle] [default carrier:]: Receives 3D skeletons to visualize.

Output Ports

Services
