Description

This module redirects the robot's gaze toward a keypoint of the skeleton in the scene. By default the skeleton keypoint is the head. It sends commands to cer_gaze-controller.

Parameters

  • –auto-start: Enable autonomous mode.
  • –period: Periodicity of the module (s).
  • –gaze-follow-T: Time constant of the gaze controller used while tracking skeletons (s).
  • –gaze-seek-T: Time constant of the gaze controller used while looking around (s).
  • –inactivity-thres: Threshold for detecting inactivity of the skeleton.
  • –virtual-mode: Enable virtual modality.
  • –robot-skeleton-name: Tag of the robot's skeleton.
  • –line-filter-order: Order of the median filter applied to the estimates of the end-line.
  • –frame: Frame wrt which the skeleton is transformed (camera or world).

Input Ports

  • /attentionManager/opc:i [Bottle] [default carrier:]: Retrieves 3D skeletons from objectsPropertiesCollector.
  • /attentionManager/gaze/state:i [Property] [default carrier:]: Retrieves the current pose of the camera.

Output Ports

  • /attentionManager/gaze/cmd:rpc [rpc] [default carrier:]: Sends commands to cer_gaze-controller.

Services

  • /attentionManager/cmd:rpc [rpc-server]: service port . This service is described in attentionManager_IDL (idl.thrift)
