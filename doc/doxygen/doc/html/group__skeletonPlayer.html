Description

This module is responsible for playing back the trajectory of a skeleton, as recorded by means of yarpdatadumper.

Parameters

Input Ports

Output Ports

  • /skeletonPlayer/opc:rpc [rpc] [default carrier:]: Sends commands to objectsPropertiesCollector to add played skeleton to the database.
  • /skeletonPlayer/viewer:o [Bottle] [default carrier:]: Steams out the 3D skeleton to be visualized. To connect to skeletonViewer.

Services

  • /skeletonPlayer/cmd:rpc [rpc-server]: service port . This service is described in skeletonPlayer_IDL (idl.thrift)
