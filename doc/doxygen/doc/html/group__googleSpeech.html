Description

This module retrieves raw input from the microphone port and queries google cloud speech services and provides the transcript on a port.

Parameters

  • –name: Module's name; all the open ports will be tagged with the prefix /name.
  • –language_code: Use English US
  • –sample_rate_hertz: Use 16000 hertz for the sample rate
  • –offline: Runs the module without connecting to Google Services. Does not process the speech

Input Ports

  • /googleSpeech/sound:i [Sound] [default carrier:]: Port to receive sound from the microphone

Output Ports

  • /googleSpeech/result:o [Bottle] [default carrier:]: Sends the transcript result of the output of google
  • /googleSpeech/commands:rpc [rpc] [default carrier:]: Sends commands to the microphone device driver to start and stop the acquisition.

Services

  • /googleSpeech/rpc [rpc-server]: service port . This service is described in googleSpeech_IDL (idl.thrift)
