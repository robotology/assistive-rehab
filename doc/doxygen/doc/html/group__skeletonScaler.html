Description

This module superimposes a (pre-recorded) template skeleton on the current skeleton. It sends commands to the skeletonPlayer for loading the template skeleton and computes the roto-translation matrix which allows to align the shoulder center keypoints and the planes of both skeletons.

Parameters

  • –nsessions: Number of repetitions the recorded skeleton is played (0 means infinite)
  • –tbegin: Stream starting time computed from the time origin.

Input Ports

  • /skeletonScaler/rpc [rpc] [default carrier:]: Receives the following commands and provides replies: (notation: "." identifies a string)
    1. load skeleton file "load" "file" "context":load skeleton file. "file" name of the file containing the skeleton data. "context" context used to look for the file.
    2. set camera options "rot" pos_list focal_list :set camera options to skeletonViewer. pos_list the position of the camera in world coordinates. focal_list the focal point of the camera in world coordinates.
    3. start "run" twarp :start. twarp specifies the warping factor squeezing (dilating) in time the original stream if less than 1 (greater than 1).
    4. set skeleton's tag "tags" "tag" :set skeleton's tag. "tag" tag of the skeleton.
    5. stop "stop":stop.

Output Ports

  • /skeletonScaler/opc [rpc] [default carrier:]: Sends commands to objectsPropertiesCollector to retrieve template and selected skeletons.
  • /skeletonScaler/player:rpc [rpc] [default carrier:]: Sends commands to skeletonPlayer for playing back the trajectory of the selected skeleton.
  • /skeletonScaler/viewer:rpc [rpc] [default carrier:]: Sends commands to skeletonViewer to set camera options for visualization.
